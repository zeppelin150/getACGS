We build tools that show you what your AI is actually doing.

Cambric AI is a small team focused on one problem: AI coding assistants are powerful, widely adopted, and operate without any meaningful oversight. We're building the tools to change that.

// THE PROBLEM WE SAW

AI coding assistants like Claude Code can read any file in your project, execute arbitrary commands, and make network requests — all without an audit trail. Developers trust these tools because they're productive. But trust without visibility is a liability.

We saw this gap growing every day. Millions of developers running AI tools that operate unsupervised inside their most sensitive codebases. No logs. No rules. No way to know what actually happened after a session.

// WHAT WE BUILT

Harpax is our first product. It monitors every action your AI coding assistant takes and gives you a complete, searchable record of what happened. It can block dangerous actions in real time or run passively and just log. It's open source, self-hosted, and all data stays local.

It works today. You can install it in under two minutes.
